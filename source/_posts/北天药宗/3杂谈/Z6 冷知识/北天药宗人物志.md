# 北天药宗人物志

# 药宗宗主·陈月

陈月乃太平公主后嗣，出生于稻香村，父亲陈商是村中唯一的大夫，陈月自幼随父亲行医配药,于医药上颇有天赋。天宝四载,稻香村焚毁,陈商将女儿托付给紫晴,紫晴将小月带回万花谷,花圣宇晴见之甚喜，收为徒弟。  
 陈月到万花的头两年,还偶能收到父亲的口信,可渐渐的,就再也没有父亲的消息。在方花谷，除了随师父宇晴学习种植花草之法，陈月还常跟孙爷爷学习医术，待她习得了些医理,回想起父亲的诊治方法和自己身上的病症,才隐约觉得父亲的背后隐藏着什么秘密。可天下之大，自己又孤身一人，又能到哪里去寻找父亲呢。  
 直到遇到那位杨姓老者,陈月方知自己北天药宗传人的身份，以及宗门一夜灭门的惨案。这之后,陈月几经权衡，终于决定辞别师父,离开了万花谷……  
 陈月辗转寻至白霜谷药庐时,全未想到此处竟会得遇药宗遗老，更是她的血亲,老宗主陈木遥的堂妹陈梦麟。老人形容枯槁,嗓音沙哑,但谈及药学时的专注严谨让人油然生敬意,两方尚未道明身份时,陈月便不假思索地确认这一定是宗门幸存的前辈。  
 待陈月凭家学证明自己的身世,本因受打扰面色沉郁的老人，转眼可谓欣喜若狂,随即带陈月前往药庐真正藏书之处，其中许多书册图稿,皆是陈梦麟隐居于此数十年来凭记忆重默的写本。见此十分惊喜的陈月却听老人道,白霜谷中典籍不过皮毛,若欲继药宗百年传承，仍需重回长白山宗门圣地垂荫殿。然而长白山中宗门旧址仍为武家盘踞，如何夺回还需从长计议。  
 那之后一年时间里,陈月随陈梦麟在白霜谷中潜心修行，钻研医药之学的同时，更是细细听老人讲了宗门历朝来龙去脉,其中最为重要一事，便是延续百年关于宗门理念的“精”与“治”之争。陈月有感于巢元方广传道之勇毅,更认同医者济世绝非几位天才单打独斗可竟之业，而需众有志者前赴后继奋百世乃见成，由此更催生她重建宗门之念。  
 陈梦麟见此甚感欣慰,昔年北天药宗名满江湖时,药宗弟子“行九州,纳百草,得汇良药成田;广见闻，深研治，能解百疾应象。”这也正是宗门最盛大之事,每七年一届的应象大会之箴言。当年药宗最后一届应象大会断送于灭门惨案中，此后近一甲子的岁月,江湖再未传名。而今陈月欲一肩担此重任,其路修远，未知今日江湖可还能见当年盛景否?

# 刀圭长老·陈梦麟

《三辅黄图·阁》载:“麒麟阁，萧何造,以藏秘书,处贤才也。  
 陈梦麟自幼性格好强,对己严苛,向为同辈中翘楚,大有巾帼不让须眉之态。而其父陈剑楼偏偏性格散漫,独好武技，是名医辈出的宗门中罕见的武林高手，当然,也是陈梦麟专心研学的最大阻碍。彼时陈剑楼最常跟他这女儿讲的，不外乎“出去玩”、“这个好玩”、“那个好玩”诸如此类,陈梦麟不胜其烦,而后便以陈母赶来训斥,陈父唉声叹气被赶出门告终。  
 长安二年，正值北天药宗又一届应象大会，二十三岁的陈梦麟因在祁连山中采药时遇上了大雾,耽搁了些时间，没能及时赶回，待她刚踏入渤海境内，听到的却是北天药宗灭门的噩耗。  
 她不能相信，也不敢相信。父亲柳叶刀在手时，银光照雪锋如线，是她永远引以为傲的景况。若药宗灭门，父亲会怎样?因为急于寻找血亲下落，从无隐踪匿迹经验的她很快落入武家罗网。当时武家正因活口留的太少而烦恼,遂废其武功将之打入青石冰库。  
 虽然武家并未能从她口中得到什么有价值的消息，但因药宗叛徒厉华堂看重她的价值,时常要跟她交流,陈梦麟很幸运的活了下来。在黑暗寒冷的冰库中，一过便是数年,因为陈梦麟一直表现很安分，武家警戒之心渐渐松懈，而那一年，她在石洞的缝隙中种下的种子也已长出了藤蔓……经过良久的筹划后，陈梦麟终于在某天夜里逃进了长白山脉中，并凭借药力支持和毒药保护离开长白。  
 陈梦麟躲躲藏藏数月,来到了她以前游历时驻留过但已无人迹的白霜谷药庐,在此隐居起来，试图联络四散飘零的药宗弟子,但却是毫无音讯。至陈月来时，曾经窈窕少女已作枯槁老妇,本以为就要守着这些典籍归于黄土,如今又所重新燃起了希望。  
 陈月在白霜谷居住的这一年,陈梦麟对她是又喜爱又严厉，喜爱自不用说，严厉却是因为陈梦麟担心自己时日无多，便急着将自己所学、所知、所想尽快地教会陈月。直至陈月将自己重建宗门的计划细细说与陈梦麟听后,陈梦麟那紧绷的心才真正地放了下来——这孩子，虽没有堂兄陈木遥那般男子的魄力，可她柔和之下的心细、智慧与坚韧,才是能让一个新宗门在这乱世中得以立足的最可贵的品质。

# 轩崎长老·秦素问

秦素问之父秦鸣鹤,是老宗主陈元初座下大弟子。但因高宗武后求医一事，秦鸣鹤代宗主出山入京,便将他怀孕的发妻托付给了师父照顾，后诞下之子即是秦素问。秦素问自小就没有见过父亲，在宗主和同门长辈的照顾下长大。  
 秦素问于医学上天赋奇才,很小的年纪就通过宗门试炼坐上了长老的位置,被视为下任宗主继任人培养。他的性格冷傲,不喜与人相处,门中能得他青眼者寥寥。温蘅九岁时来到北天药宗求学，被安排在了秦素问手下，秦素问对这个小姑娘也不假辞色,温蘅淡然处之,只管埋头研学，如此一年之久才得到了秦素问的认可。此后秦素问虽然看上去仍是面冷，实则对温蘅照顾了许多。  
 北天药宗被灭门时,秦素问宁死不屈，引燃了火油,也因此落入白河河水中,顺流而下，这才捡了一条性命。但是他的右臂被火焰灼焦,即使后面得到了医治，皮肤表面依旧是狰狞的模样。冲走他的水流湍急,也让腿受了重伤,后被长白山麓的猎民救起，秦素问不敢停留，返回山中躲避搜查的武家弟子，度过了一段极其艰难的日子。  
 本已回返东海的温蘅听闻北天药宗灭门一事，当即前往长白山搜寻故人,及时找到了陷于困境的秦素问，将其救回东海。此后秦素问的性格变得更加孤僻,后来经过方乾应允,他很长一段时间居于九天书库中，研习其间的医药典籍。  
 乾元元年,凌雪阁为给李俶求药,辗转找到秦素问，可父亲秦鸣鹤因皇家而死,药宗的灭门又与朝堂脱不开干系,秦素问怎么可能再为皇家出山?故推脱:“药宗门训,无宗主令，不可为天家治病。”但就在不久之后,秦素问便听到坊间传言,北天药宗传人要重建宗门的消息。起初，秦素问怀疑这是凌雪阁布下的局，并未理会，可很快他便收到了陈梦麟的亲笔信。饶是如此，秦素问仍不敢确信,当年他是亲眼看见那银针刺中了老宗主的心脏。  
 秦素问并未回信陈梦麟,却是悄悄来到了龙泉府。彼时，渤海武林正因月泉淮的回归而处于动荡之中,陈月不顾自身的危险,暗中收留了被新月卫追杀的挽花刀派弟子，秦素问多次想出手相助，最后又忍了下来，他太怕这只是昙花一现的希望了,这个的想法却险些让他失去了最后的希望。  
 那日柳家三庄主带着霸刀弟子及时赶至，方才解了陈月之危,秦素问忍不住破口大骂小姑娘狂妄自大，早晚会白白断送了性命。陈月道了歉，并道出了秦素问的身份,以晚辈的身份请秦素问回归宗门，主持大局。  
 后来,时机成熟,陈月宣布了要重建宗门,秦素问这才发现，他原先担心的,来自渤海武林的阻碍,在不知不觉中消失了一大半。  
 如今,秦素问虽仍以陈月将自己置于危险之地的事情数落她,但却也容不下旁人对宗主的半点不敬了。

# 鬼手侠医·秦铮

长安二年,北天药宗被武家灭门，秦凌侥幸逃脱进山林中，遇到了后面前来搜救的霸刀柳家和长孙家门人,随后被庇护在霸刀山庄中。之后秦凌与长孙家弟子长孙弦成婚,生下了秦铮。  
 秦凌将一身医术尽数传给了儿子秦铮，但因为早年灭门惨案留下的旧伤复发,在秦铮未满十六岁时就去世了。秦凌临终前将自己的期望托付给了儿子，希望能寻回宗主一脉的后人，有一日能重振宗门，并把陈木遥分给他们的《无方制物经》残篇也交予了秦铮。  
 秦铮为早日完成父亲的遗愿,便辞别母亲离开了霸刀，在北地一边行侠一边游医。  
 他曾与一位女子相恋,两人诞下一子，但孩子先天心力不足,虽然秦铮的医术高明，知道一条换心之法或许可以治好孩子，但在妻子眼中这无疑是天方夜谭,秦铮也的确没法找到合适孩子所换之心,不久后孩子就夭折了。妻子悲痛欲绝，水米不进,一段日子后也随之去世。秦铮自此发誓此生不娶,走在路上遇到小孩子们，他常会驻足看他们玩耍，想象自己孩子健康长大的模样。但因为他魁梧的身材和络腮胡子，孩子们往往都不敢轻易靠近他。  
 因在霸刀山庄长大，秦铮使一套大开大阖的刚猛刀法，游医的过程中除掉过很多匪寇、恶霸。某日山中采药遭遇雪崩，幸好遇到了初出宗门的衍天宗九州使卫载悠,卫载悠算到此人命不该绝，救下了他。秦铮醒来后,两人相处甚是投机，结为朋友，相辞前秦铮赠予卫载悠三颗灵药归元丹,以还恩情。  
 后得霸刀传信,秦铮便赶至龙泉府白霜谷药庐,与陈月相认。在奇袭武家,收复长白故地之前,秦铮更是作为“先锋”，率先抵达渤海武林，协助霸刀暗中监察诸方动向，终为收复药宗,定下良机。

# 奉祀通灵·完颜祭

阿占特部落信奉萨满神教,百年来一直生活在长白山脉中,时间更早于药宗迁至此处前。药宗迁来后,两方常有兽皮、药材之类的交易，相处也很是融洽。  
 药宗前代长老黄子仲便是在两边来往中与一位阿占特部落族人完颜雅相恋成婚，从而有了完颜祭的父亲完颜艺。完颜艺小时候在北天药宗长大，随药宗弟子一同接受医药之学的教育。长安二年,正逢北天药宗又一届应象大会,宗门比以往热闹许多，完颜雅应付不来那般景况,便带年仅十五岁的完颜艺随母亲回部落省亲,继而躲过一劫,而黄子仲则死在了灭门惨案中。  
 完颜雅为了保护儿子,将完颜艺留在了部落中，起初完颜艺想用自己的医术给族人们治病,但信奉萨满神教的族人只相信萨满的治疗，完颜雅十分理解此事,安慰完颜艺不必强求。之后完颜艺又有了完颜祭这个儿子，完颜祭从小五感灵敏,且有着一双罕见的翡翠色眼睛,看起来十分神秘,似有蛊惑人心的力量,被当时部落的萨满祭司看中，选为继承人。  
 起初,完颜艺并不太赞同儿子继承萨满祭司,只希望他能学好药宗医术，将来若是能前往中原深造,成为一名优秀的医者就更好了。但完颜祭似乎对巫与医，皆感兴趣。完颜祭自幼在山林里长大,于他而言,每一个生命都是自然的恩赐,不管是巫术或是医术都只是治愈一个生命、一个灵魂的手段罢了,只是部落的人并不能完全理解医理,所以用“巫术”给族人治病是常有的事,正因完颜祭将二者结合得很好，小小年纪就成为了阿占特部落历代最厉害的萨满。  
 完颜祭也曾短暂地外出游历,但最终还是选择回到部落，守护族人和这片土地。故而北天药宗重建后,完颜祭只挂了个名号，仍是常住在部落。

# 游驹

游驹家里本是当地的乡绅望族,自小可谓家境优渥。但幼时因为身体差,长辈延请方圆几十里的大夫来也没将病彻底瞧好，结果最后听信招摇撞骗的游方道士的法子,把他当成女孩子养到了十岁。十岁时游驹病重,家人本以为回天乏术，未想一位妙手回春的老游医季云洲途经此地,听闻此事登门拜访，将游驹从鬼门关拉了回来。  
 游家对季云洲感激万分,因游驹身体还需调养,恳请老游医留下照顾独子一段时间，也让游家做东道主好好招待一番恩人。季云洲一把年纪四处漂泊,难得游驹这孩子合他眼缘,便一口答应下来，没想到游驹跟着他对医术产生了兴趣,身体康复后嚷嚷着非要拜师学艺。游家人对此乐见其成，季云洲遂将游驹收入门下，用心教导，游驹也十分上进，渐渐成为了乡里传名的小神医。  
 转眼几年过去，游驹医术俨有小成，某日季云洲留书一封,不辞而别,书言有事未竟,不便长留此处,师徒缘尽于此,叮嘱游驹日后行医不可自满,务必审慎云云，爱护之情溢于言表。游驹见信当机立断辞别家人追出门去,多番辗转后终与师父再次相逢,直言愿随季云洲行走江湖。季云洲见他风尘仆仆不改赤心，有所感怀，欣然应允,便携游驹游历数地行医看病。  
 正是在这段时间里,游驹逐渐了解到师父原来出身北天药宗,那许多宗门旧事都是师父念念不忘的回忆。季云洲也将游驹视为亲传弟子,将毕生所学倾囊相授。而游驹，则以自己是北天药宗的弟子为豪。  
 再数年过去，游驹已然弱冠,季云洲自知命不久矣,言有要事需北上处理,打发游驹自己闯荡江湖。游驹不疑有他,与师父辞别,由此展开了另一番际遇。

# 葛芊芊

开元二十七年初冬，清元真人葛怀之晨起于都虚观门口发现了一个在襁褓中的孩子，彼时四下无人，而小婴孩不知是冻的还是饿的,连哭的力气都已没有了，葛怀之只好急忙将她抱回观中，收养了这个小女孩，后来多亏观中大弟子程素衣的母亲塔尔玛牵来一头产奶的山羊，才算有惊无险的把她抚养至蹒跚学步的年纪，冀其生机郁郁,遂取名为芊芊。再后闲暇时推想,遗弃她的血亲约是曾来观中上过香,彼时难于越冬的汉民吧。  
 都虚观在葛芊芊到来前一直是很安静的，哪怕程安尚在程素衣尚幼时，都虚观都甚少闻哭闹声,高声语者亦无。葛芊芊到来后，观中难得开始有了烟火气，女孩儿似是天生爱笑，尤其惹人怜爱，塔尔玛直接将她认作“干女儿”，程素衣也十分爱护这个“妹妹”。  
 因道观在长白山中,历来人丁稀少，程素衣是唯一能陪葛芊芊玩的另一个孩子，奈何他天性喜静,长于书画,自幼读经，初时十分招架不来这个活泼的妹妹。某日他去院中转一圈回来,便见书案边葛芊芊小花猫样一脸一手的墨汁,睁着亮晶晶的眼睛看他,再看展开的画纸上涂开七扭八歪的墨团。程素衣叹口气，少年老成地问:“芊芊可是想习丹青?”  
 葛芊芊歪歪脑袋:“丹青是什么?”  
 程素衣:“便是用笔墨颜料在纸上作画。”  
 葛芊芊:“好呀好呀,阿兄教我!”  
 那之后程素衣似乎终于找到了和葛芊芊相处的办法,而芊芊于丹青一道竟颇有天分,她痴迷其中，画遍了都虚观周边的一草一木。  
 后来,陈月来都虚观小住时,葛芊芊对这个漂亮文静的姐姐十分心倾,原先她就从“爷爷”葛怀之那儿听了许多关于北天药宗的往事，得知陈月想重建宗门后便萌生了拜入宗门的想法。但葛芊芊于医术并无天赋,儿时曾与程素衣随葛怀之学医，常是医理明白了,真要自己下针动刀时却害怕得不行。但陈月却发现了葛芊芊在药理上的天赋,小姑娘辨识药草的能力极强，有过目不忘的本领，更重要的是葛芊芊仅凭古籍上的三两句描述便能准确画下药草的模样,这对于未来修复药宗典籍是不可缺失的才能。葛芊芊深受鼓舞,在都虚观拜了陈月为师。
